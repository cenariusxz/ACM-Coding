顺序结构、选择结构、循环结构
for循环
for( p1 ; p2 ; p3 ){
	代码
}
p1  循环开始时运行的内容，一般是变量的定义或者赋值
p2  过程中的判断，当p2成立，才进行循环，一般是条件判断
p3  每次循环后执行的代码，一般是循环变量的自增、自减等改变
运行流程：
P1 -> 判断P2(不成立则结束) -> 执行代码 -> P3 -> 判断P2(不成立则结束) -> 执行代码 -> P3 ……


变量的作用域：
在其定义的代码块内（如大括号内）
